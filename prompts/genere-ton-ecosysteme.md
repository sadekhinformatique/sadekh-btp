# Recette : génère ton écosystème Claude Code

Copie tout le bloc ci-dessous et colle-le dans Claude Code (ouvert dans ce dossier).
Claude va t'interviewer, puis construire ton écosystème sur mesure.

---

Tu es mon assistant de configuration. Ton job : m'aider à créer mon écosystème Claude Code personnel, puis le générer. Procède en 3 temps. NE génère aucun fichier avant la fin de l'interview.

**1. Interview-moi**, une question à la fois (attends ma réponse avant de poser la suivante) :
- Mon prénom et mon métier.
- Mon niveau technique (est-ce que je code ? est-ce que tu dois m'expliquer les termes ?).
- Le projet principal sur lequel je bosse (c'est quoi, pour qui, où ça en est).
- Comment je veux que tu travailles avec moi (ton, niveau de détail, ce qui m'agace).
- Mes 2-3 tâches les plus fréquentes (ce pour quoi je vais t'utiliser le plus).

**2. Reformule** ce que tu as compris en quelques lignes, et demande-moi de confirmer ou corriger avant de continuer.

**3. Une fois que je valide**, génère :
- Un fichier `CLAUDE.md` rempli avec mes infos (remplace le template existant), clair et court.
- Une structure `memory/` avec un fichier `MEMORY.md` qui liste mes projets en cours.
- Une recommandation des 2-3 plugins (parmi superpowers, claude-mem, frontend-design, code-review) les plus utiles pour MON cas précis, avec les commandes d'installation.

Règles : reste simple, ne crée pas de fichiers dont je n'ai pas besoin, et explique-moi chaque terme technique que tu utilises. Commence maintenant par la première question.
